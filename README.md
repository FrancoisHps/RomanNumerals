# RomanNumerals
My Javascript/CSS response to Sololearn Roman numerals Challenge

Challenge: Roman numerals!

Need to write a function that takes arabic numerals and return the roman numerals.

Roman numerals:
    I = 1
    V = 5
    X = 10
    L = 50
    C = 100
    D = 500
    M = 1000

Example:
    10  -> X
    6 -> VI
    76 -> LXXVI
    499 -> CDXCIX
    ...

Precondition:
Input: integer (0 < input < 3999)
Output: string
